# whack-a-mole-game
Classic Whack-A-Mole re-created in Python using Pygame

# Requirements
  1) Python 3
  2) pygame

# How to run
## Terminal:
1) Run command:     `python3 main.py`

## IDLE:
1) Open file: main.py
2) Run the module

# Game Rules
  1) Click the mole character to whack it and earn points

# Files
  1) main.py: Game runner
